fefemama 3.0

This is the newest version of my Blog. It started, years ago, with Wordpress 1.x and went on to a custom, C99 based blogging system. But that became boring, too. And I decided to make something totally different.

So I took python and mongodb and built a blog. But HTTP all over again sounded so mainstream and I went instead with ... Telnet.

But Python was not my thing and so I took Go (golang) and remade it again. And added a http interface again with web.go :)

You can take a look at the telnet frontend by telnetting to port 1337:

% telnet fettemama.org 1337

Connect, type "help" and explore. (Most blog posts are in German though.)

If you got curious about the blogging software just go to GitHub and take a look at it.

Contact me via email if you've got something to say. Or leave a comment on the blog.
